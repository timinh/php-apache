# php8.1-apache-for-symfony6
A php 8.1 + apache dockerfile ready for symfony 6

### Contains the following extensions : 
 - pdo
 - intl
 - mysqli
 - gd
 - ldqp
 - gettext
 - calendar
 - ctype
 - session
 - dom
 - pdo
 - pdo_mysql
 - curl
 - zip

### Contains the following utility programs : 
 - nano
 - composer
 - git
