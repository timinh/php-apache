# php-apache
A php apache dockerfile ready for symfony, laravel or any php project.

### Contains the following extensions : 
 - pdo
 - intl
 - mysqli
 - gd
 - ldap
 - gettext
 - calendar
 - ctype
 - session
 - dom
 - pdo
 - pdo_mysql
 - curl
 - zip

### Contains the following utility programs : 
 - nano
 - composer
 - git
 - cron
